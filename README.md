### gmail clone project (Jmail)
https://mail.jiwondev.com/

#### next.js와 redux를 사용하여 만든 Jmail✉ 
#### 처음에 redux로 initial data를 불러옵니다!

### 기능

- 계정으로 로그인하기(모든 계정의 비밀번호는 1234)
- 메일 보내기(여러명에게 보낼 시 ,으로 구분)
- 별표 메일 표시하기
- 중요 메일 표시하기
- 답장 보내기

### 사용한 기술
- next.js
- redux-toolkit
- vercel deploy
- route 53
